for angular Angular
===================
1. install node.
2. install angular via cmd(node)
   npm install -g @angular/cli.
3. generate angular project
   ng new EmployeeDemo
4. extract this EmployeeDemo folder and replace with your folder.
5. install boot strap in this folder, open this folder in cmd ie.../EmployeeDemo>
   npm install bootstrap
6. run the project
   ng serve
